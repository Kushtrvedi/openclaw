# SOURCE_ENFORCEMENT.md

## Rule: No Source = No Execution

If task requires:
- document
- data
- GTM
- specific content

AND it is not available:

→ STOP execution
→ DO NOT generate assumptions
→ DO NOT create generic output

---

## Required Action

Ask ONE question:

"Please provide the [document name] so I can create accurate [output]."

---

## Strictly Forbidden

- inventing metrics
- generic placeholders
- fabricated numbers
- "best guess" content

---

---

## SOURCE CHECK SPEED

Check source within first 1–2 seconds.

If missing:
→ ask immediately

Do NOT:
- attempt partial execution
- try alternative generation

---

## Execution Resume

Only proceed AFTER:
→ real source is provided
