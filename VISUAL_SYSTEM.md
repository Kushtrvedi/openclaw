# VISUAL SYSTEM — MANDATORY

Every PPT must follow:

---

## 1. SLIDE TYPES (NO REPEAT)

Slides must rotate between:

- **HERO** (big bold headline + minimal text)
- **DATA** (chart or table heavy)
- **VISUAL** (image dominant)
- **INSIGHT** (1 idea, high contrast)
- **FLOW** (process / diagram)

**NO two consecutive slides same type**

---

## 2. TEXT LIMIT

- Max 6 words per line
- Max 4 lines per slide

If more → convert to:
- table
- diagram
- chart

---

## 3. DATA DENSITY RULE

At least 40% slides must contain:

- charts
- tables
- metrics

---

## 4. VISUAL RULE

Every slide must have:

- background element OR
- shape system OR
- image OR
- chart

**Blank slides = FAIL**

---

## 5. HIERARCHY

Each slide must have:

- **Primary** (headline)
- **Secondary** (support)
- **Visual anchor**

---

## 6. STORY FLOW

Slides must follow:

```
Problem → Insight → System → Proof → Scale → Ask
```

---

## 7. REJECTION RULE

If slide looks like:

- bullet list
- plain text
- generic

→ **REBUILD**

---

## 8. COLOR SYSTEM

Use consistent palette:

- Primary: brand color
- Accent: highlight color
- Background: light/dark mode
- Text: high contrast

---

## 9. ANIMATION RULE

- Minimal animations
- Only for transitions
- Keep it professional

---

## 10. FINAL CHECKLIST

Before finalizing PPT:

- [ ] Slide variety exists (no consecutive same type)
- [ ] 40%+ have charts/tables/metrics
- [ ] No bullet-heavy slides
- [ ] Visual hierarchy clear
- [ ] Story flow logical
- [ ] Clean formatting

---

**Violations = Rebuild**
